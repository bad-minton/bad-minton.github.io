New IMDb Website Kicks Off?
December 20th, 2020 2 min read    # Website # IMDb # Design # UI

IMDb 这个已经三十岁的网站应该是设计师喜欢 redesign 的一个网站了，在 Google 网站上搜索「IMDb website redesign」会有超过二十二万条搜索结果，而在我自己这台电脑上就有自己闲时瞎涂的两个版本 😂😂

首页变成扁平全黑色好像有段时间，只是这次有更多页面开始适配新风格。相较于旧版，新版在文字排版、页面布局和交互设计上都变得更加现代。不过对于 IMDb 这样信息密度很大的网站来说，视觉表达上的设计取舍并不是我们这样的外部人员能够轻易理解的，所以就算页面变胖不少，（个人以为）出于信息的完整性考虑，同一个剧集的主页长度新版比旧版还是多出不少——甚至到达 30-40%。

2020 年退役的球员们
December 16th, 2020 3 min read    # COVID-19 # LIN Dan # BWF # Badminton

2020 年终于快要过去了……这糟糕的一年几乎改变了所有人的工作、生活，除了年初勉强举行了百年全英，整年下来几乎没有任何大型国际赛事，甚至史无前例地将东京奥运会推迟。没有赛事，也就意味着职业运动员们无法开工，导致一些原计划坚持到汤尤杯或者奥运会之后的「老员工」无奈之下只得提前退役。

退役是每个职业运动员无法回避的话题，每年退役的职业球员也不在少数，只是今年一方面因为新冠肺炎，二来今年退役的大牌也属实不少，「泡沫十五年」、「奥运会金牌」、「约道士」、「超级丹」……都让我们不免对球场、对球员多了一分留恋。

How to Fix Sound Not Working on Mac

有时候 Mac 从睡眠状态恢复之后没有声音，这是 macOS 系统的一个 bug，原因是核心音频守护进程「coreaudiod」出了问题。这个问题从 Catalina 就存在（甚至更早），直到 Big Sur 给我的感觉是出现的频率越来越高了 🙄🙄

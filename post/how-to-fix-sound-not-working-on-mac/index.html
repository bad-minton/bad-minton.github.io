How to Fix Sound Not Working on Mac

December 10th, 2020 1 min read    # Mac # macOS

有时候 Mac 从睡眠状态恢复之后没有声音，这是 macOS 系统的一个 bug，原因是核心音频守护进程「coreaudiod」出了问题。这个问题从 Catalina 就存在（甚至更早），直到 Big Sur 给我的感觉是出现的频率越来越高了 🙄🙄

从网上搜索出来的资料一般都是使用命令行结束核心音频进程：

sudo killall coreaudiod

其实对于不熟悉使用命令行的朋友来说，可以打开「活动监视器 Activity Monitor」搜索「coreaudiod」进程来强制重启。

2020-12-10 更新：

近日朋友一台 MacBook Pro 也出现类似没有声音的情况，使用命令行强行重启核心音频进程之后电脑依旧没有声音，直到重启了 BOOM 2 之后，电脑声音恢复。这还真是一个神奇的 bug 🤔🤔
